# Common libs

[![](https://jitpack.io/v/naixx/common-libs.svg)](https://jitpack.io/#naixx/common-libs)



Add it to your build.gradle with:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
and:

```gradle
dependencies {
    compile 'com.github.naixx:common-libs:{latest version}'
}
```
